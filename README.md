# Lunch Bot

This repository is for scraping lunch menus and sending them to an MS Teams channel.

## Contribute

It's quite simple:

1. Install [Git](https://git-scm.com/downloads).

    ```shell
    winget install -e --id Git.Git
    ```

1. Install the [Docker](https://docs.docker.com/get-docker/) engine.

    ```shell
    winget install -e --id Docker.DockerDesktop
    ```

1. Install [Visual Studio Code](https://code.visualstudio.com/download).

    ```shell
    winget install -e --id Microsoft.VisualStudioCode
    ```

1. Install the VS Code *Dev Containers* extension.  
   [VS Marketplace link](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

    Run in VS Code:

    ```shell
    ext install ms-vscode-remote.remote-containers
    ```

1. Clone this repository and open it in VS Code.

1. Click the `Reopen in Container` button in the bottom-right corner.

1. Create and switch to your new Git branch:

    ```shell
    git checkout -b new-branch
    ```

1. Enjoy contributing!
